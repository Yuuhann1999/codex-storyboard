# Upstream

This project is an independent local extension of
[`Yuuhann1999/codex-storyboard`](https://github.com/Yuuhann1999/codex-storyboard),
initially based on commit `ac9057dee3a903eb211d8399a439ae9992e7656a`
(`0.5.4`). Preserve the upstream MIT license and attribution when syncing code.

The development plugin is named `storyboard-workbench` so it can coexist with
the installed upstream `codex-storyboard` plugin. Do not edit Codex plugin cache
directories directly.
